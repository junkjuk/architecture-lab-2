# Лабараторна робота №2 з архітектури ПЗ

# Postfix calculator

## Прапорці

Зчитування з командного рядку:
```bash
-e "вираз_для_обчислення"
```
З файлу:
```bash
-f "ім'я_файлу"
```
Запис результатів в файл:
```bash
-o "ім'я_файлу"
```
